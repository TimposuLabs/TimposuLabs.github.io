# Java Generic

## Isi Materi

Di materi ini, kita akan belajar fitur dalam bahasa pemrograman java yaitu Java Generic. Adapun materi yang akan dibahas pada tutorial Java Generic adalah:

* Generic Class
* Generic Method
* Generic Invariant
* Generic Covariant
* Generic Contravariant
* Generic Wilcard
* Generic Bounded Type
