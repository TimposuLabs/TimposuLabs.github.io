# Java Dasar

## Isi Materi

Di materi Java Dasar, kita akan belajar tentang konsep dasar Pemrograman. Adapun materi pokok yang akan dibahas pada Tutorial Java Dasar adalah:

* Pengenalan Java
* Instalasi Java
* Tipe Data
* Operator
* Statemen Kontrol (pemilihan, pengulangan, peloncatan)
