## 1️⃣ Clean Project

```
mvn clean
```

## 2️⃣ Compile

```
mvn compile
```

## 3️⃣ Running Unit Test

```
mvn test
```

## 4️⃣ Package Project

```
mvn package
```

Hasil build project berada dalam direktori `target`, dengan ekstensi `.jar`.
