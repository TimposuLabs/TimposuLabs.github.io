## Program Pertama

1. Membuat function main

```kotlin
fun main() {
    println("Halo Bro")
}
```

Output

```
Halo Bro
```

2. Membuat function main dengan argumen

```kotlin
fun main(args: Array<String>) {
    println(args.contentToString())
}
```
