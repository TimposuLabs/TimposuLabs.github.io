title: 'Type Data Char'
---

Data karakter/character (huruf), di Kotlin direpresentasikan oleh tipe `Char`. Untuk membuat data `Char`, di Kotlin kita bisa menggunakan tanda `'` (petik satu).

```kotlin
fun main() {
    var o: Char = 'O'
    var g: Char = 'G'
    var u: Char = 'U'
    var d: Char = 'D'

    print(o)
    print(g)
    print(u)
    print(d)
}
```
