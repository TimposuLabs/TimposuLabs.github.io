# Kotlin Dasar

## Isi Materi

Di materi Kotlin Dasar, kita akan belajar tentang konsep dasar Pemrograman. Adapun materi pokok yang akan dibahas pada Tutorial Kotlin Dasar adalah:

* Pengenalan Kotlin
* Tipe Data
* Operator
* Statemen Kontrol (pemilihan, pengulangan, peloncatan)
